## Semana 3 e 4 

## Objetivo:
A terceira e a quarta semana tiveram como objetivo aprimorar tudo aquilo que foi desenvolvido nas semanas 1 e 2, sanando as principais dificuldades da monitoranda, como os comandos Git e o dominio da linguagem em Javascipt.

## Conteúdo Abordado:
Javascript, comandos Git e propriedades do GitHub

## Projetos/Exercicos Desenvolvidos:
A monitoranda realizou exercicios lógicos em Javascipt e realocou cada atividade em uma pasta referente à semana da monitoria. A organização do repositório Hello World foi feito graças à prática dos comandos Git. 

## Tecnologias Usadas:
A maioria dos projetos das duas semanas foi usando somente o VS Code e a linguagem do Javascript.

## Como Rodar (extensões, comandos...):
Para rodar os códigos desta pasta foram utilizados o editor de código VS Code, a linguagem Javascript, o Node.js e o Code Run
